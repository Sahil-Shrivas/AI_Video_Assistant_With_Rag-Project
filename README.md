# 🎥 AI Video Assistant RAG

🚀 A powerful AI-powered web application that allows users to input YouTube videos and ask questions based on the video content using a **Retrieval-Augmented Generation (RAG)** pipeline built with Streamlit.

---

## 📌 Project Overview

This project extracts knowledge from YouTube videos (via transcript/audio processing), converts it into embeddings, and enables intelligent question answering using an LLM-powered RAG system.

✨ It demonstrates:
- Retrieval-Augmented Generation (RAG)
- YouTube transcript / audio processing
- Vector database search (ChromaDB / FAISS)
- LLM-based question answering
- Interactive Streamlit UI

## 🎯 Features

✅ YouTube video input support  
✅ Transcript / audio-based text extraction  
✅ Intelligent text chunking  
✅ Vector embeddings for semantic search  
✅ RAG-based question answering  
✅ Fast and interactive Streamlit UI
